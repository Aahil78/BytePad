# BytePad
BytePad is a 3x3 Handwired Macropad with 2 EC11 Encoders, and a 0.96" OLED.

I decided to make this as I found it very inconvinient to keep changing apps or doing a dozen of clicks to change the volume, so i designed a macropad to fix these issues.
